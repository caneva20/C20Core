# Installing

* Add the [jitpack](http://jitpack.io) repository:
  ```xml
  <repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
  </repositories>
  ```

* Add the dependency:
  ```xml
  <dependency>
      <groupId>me.caneva20</groupId>
      <artifactId>c20core</artifactId>
      <version>VERSION</version>
  </dependency>
  ```
